# Infineon-EVB-Remote-Control-Board
Infineon 900V EVB Control firmware
Features ACZVS, DCZVS and DyR platforms
Compatible with unidirectional devices

To access control, run python main.py
Requires:
-900V BDS EVB (FOR POWER CONVERSION CONTROL)
-MCU CONTROL BOARD
-12V SUPPLY
-RS-485 DIFFERENTIAL CONNECTOR (A+/B-/GND CONNECTIONS)
